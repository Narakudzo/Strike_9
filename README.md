Работает в Web

Для запуска на Web нужно установить emsdk и ввести команды:
git clone https://github.com/emscripten-core/emsdk.git
cd emsdk
git pull
./emsdk install latest
./emsdk activate latest
./emsdk_env.sh
./emsdk && http-server
